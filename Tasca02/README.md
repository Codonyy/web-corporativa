
# T02: Missió Apache — Desplegament Multidomini i Segur

## Breu descripció

Configuració d’un servidor web **Apache** en Ubuntu Server capaç d’allotjar múltiples portals web en un únic servidor, garantint seguretat, rendiment i bones pràctiques professionals abans del desplegament final al núvol.

---

# Introducció

Les prediccions dels assessors de la incubadora s’han confirmat: l’empresa ja disposa del seu primer client real relacionat amb infraestructura web.

**Nexus**, una nova empresa de formació situada a Mataró, ha contractat els serveis tècnics per desplegar i gestionar la seva infraestructura web corporativa.

En aquesta fase inicial, l’objectiu és construir una base **estable, segura i escalable** abans de migrar definitivament els serveis a un entorn cloud.

---

# Requisits del client

El client estableix els següents objectius tècnics:

- Allotjar **dos portals web diferenciats** en un únic servidor.
- Garantir comunicacions segures mitjançant **xifratge SSL/TLS**.
- Assegurar un rendiment òptim utilitzant protocols web moderns.
- Optimitzar recursos mantenint una única infraestructura.

Els portals web són:

- Portal de l’agència de disseny.
- Portal de l’acadèmia de formació.

---

# Objectiu tècnic

Com a tècnics especialistes, cal configurar un servidor **Apache** sobre **Ubuntu Server** que compleixi tots els requisits del client aplicant criteris professionals d’administració de sistemes.

---

# Entorn de treball

Abans del desplegament final en producció:

1. El projecte es desenvolupa en una **màquina virtual local**.
2. Es realitzen proves completes de funcionament.
3. El client valida la solució.
4. Posteriorment es desplega en un **VPS a Internet**.

---

# Descripció de l’encàrrec

El projecte consisteix en la instal·lació, configuració i securització avançada d’un servidor web Apache.

Tot el procés ha de quedar documentat dins d’un **informe tècnic**.
