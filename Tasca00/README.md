## 📘 T00: Presentació del projecte

### 📌 PART 1 — Presentació del projecte i objectiu

Aquest projecte forma part del **Projecte Intermodular**, on es treballaran diferents activitats relacionades amb la gestió de sistemes informàtics, serveis de xarxa, administració de servidors i bones pràctiques professionals dins d’un entorn real d’empresa.

L’objectiu principal és aplicar de manera pràctica els coneixements adquirits durant el curs mitjançant:

- Configuració de serveis informàtics
- Administració de sistemes Windows i Linux
- Virtualització d’infraestructures
- Documentació tècnica professional
- Treball col·laboratiu amb control de versions (Git)

El projecte simula situacions reals d’una consultoria IT, on cal analitzar necessitats, implementar solucions i documentar correctament tots els processos.

---

### 🎯 Objectius del projecte

- Aplicar coneixements tècnics en entorns reals simulats.
- Aprendre a documentar projectes informàtics professionalment.
- Utilitzar Git i GitHub com a eina de control de versions.
- Organitzar un repositori estructurat i mantenible.
- Desenvolupar autonomia en la resolució de problemes tècnics.
